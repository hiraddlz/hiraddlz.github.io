## AI Alchemy – Open-Source AI Toolbox
**Dec 2024 – Mar 2025** | [Live Demo](https://github.com/hiraddlz/ai-alchemy)

A comprehensive suite of AI-powered productivity tools designed to make AI accessible to everyone.

### Features:
- **YouTube Video Chat**: Extract and query video content directly
- **Resume Matcher**: Intelligent resume-job matching system
- **IELTS Writing Examiner**: Writing assessment and feedback tool
- **Proofreader**: Advanced text proofreading and enhancement
- **File Q&A**: Query and extract information from documents
- And 5+ more specialized tools

### Technologies:
- **Framework**: Streamlit, Python
- **AI Models**: OpenAI API, LangGraph, Transformers
- **Infrastructure**: Vector Databases, GitHub, Streamlit Cloud
- **Deployment**: Free and accessible cloud deployment

### Impact:
Provides free, accessible AI tools to users without technical expertise, democratizing access to cutting-edge AI capabilities.
